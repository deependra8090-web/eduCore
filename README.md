# 📚 EduCore

EduCore is a full-stack course selling platform where users can **browse courses**, **purchase securely**, and **access learning content** with a smooth and scalable experience.

---

## ✨ Features

### 👤 Authentication

- Secure user registration and login
- JWT-based authentication
- Role-based access control (User/Admin)

### 🎓 Courses

- Browse available courses
- Course details with pricing
- Admin can create, update, delete courses
- Structured course management system

### 💳 Payments

- Secure payments using Stripe
- Smooth checkout experience
- Payment status handling
- Reduced cart abandonment with optimized flow

### 🔍 Course Discovery

- Fast and efficient search system
- Optimized queries for quick results
- Improved user experience with structured navigation

### 📦 Backend System

- RESTful APIs for all operations
- Optimized database queries
- High performance and scalability

---

## 🛠 Tech Stack

### Frontend

- React + Vite
- Tailwind CSS
- React Router
- Axios

### Backend

- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- Stripe API (payments)
